## 미구현 — 이번에 전부 구현할 것

### 1. 도시 업그레이드 (placeCity)
- 기존 마을을 도시로 업그레이드
- 비용: 밀2 + 광석3
- 점수: 1→2 (추가 1점)
- 도시가 있는 헥스에서 자원 2개 획득 (마을은 1개)
- UI: 마을 클릭 → 도시 업그레이드 버튼

### 2. 개발카드 사용 (playDevCard)
- 턴당 1장만 사용 가능
- 구매한 턴에는 사용 불가 (turnBought 추적 필요)
- **기사 (Knight)**: 도둑 이동 + 인접 플레이어에게 카드 훔치기. usedKnights++
- **도로 건설**: 무료 도로 2개 즉시 배치
- **발명 (Year of Plenty)**: 원하는 자원 2개 선택하여 획득
- **독점 (Monopoly)**: 자원 1종류 선택 → 모든 플레이어에게서 해당 자원 전부 가져오기
- **승점 (VP)**: 즉시 +1점 (보유만 해도 적용, 공개 안 함)
- UI: 내 카드 영역에 사용 버튼

### 3. 7 굴렸을 때 절반 버리기 (discardHalf)
- 8장 초과 보유 플레이어: 절반(내림) 버림
- UI: 모달로 버릴 카드 선택
- 모든 해당 플레이어가 버린 후 → 도둑 이동

### 4. 카드 훔치기 (stealCard)
- 도둑 이동 후, 해당 헥스 인접 마을/도시 소유자 중 택 1
- 랜덤 자원 1장 훔치기
- UI: 훔칠 대상 선택

### 5. 거래 시스템
- **은행 거래**: 동일 자원 4개 → 다른 자원 1개
- **항구 거래**: 일반 항구 3:1, 전문 항구 2:1
- **플레이어 거래**: 제안 → 수락/거절 (멀티에서)
- UI: 거래 패널 (하단 또는 모달)

### 6. 승리 조건 체크 (checkVictory)
- 매 행동 후 체크
- 자기 턴에 10점 이상 → 승리
- 점수 = 마을(1) + 도시(2) + 승점카드 + 최장도로(2) + 최대기사(2)
- 승리 시 결과 오버레이

### 7. 최대 기사단 (largestArmy)
- 기사 3명 이상 사용한 플레이어 중 최다
- +2점
- 다른 플레이어가 더 많이 쓰면 이전

## 효과음 (Web Audio API)
기존 다른 게임들처럼 SFX 객체로 구현:
```js
const SFX = {
  dice() { ... },       // 주사위 굴리기
  build() { ... },      // 건설 (마을/도로/도시)
  trade() { ... },      // 거래 성사
  card() { ... },       // 개발카드 뽑기/사용
  robber() { ... },     // 도둑 이동
  steal() { ... },      // 카드 훔치기
  victory() { ... },    // 승리
  turnStart() { ... },  // 턴 시작
};
```

## TTS (딜러 보이스)
llm.cocy.io의 TTS 엔드포인트 활용:
```js
// POST https://llm.cocy.io/v1/audio/speech
// body: { input: "텍스트", voice: "alloy" }
// 주요 음성: 주사위 결과, 자원 획득, 도둑 등장, 승리 선언
```

중요한 이벤트에만 TTS:
- "주사위 X! [자원명] 생산!"
- "도둑 출현!"
- "축하합니다! 승리!"

## UI 개선
- 자원 카드를 이모지 아이콘으로 시각화 (🧱🪵🐑🌾⛏️)
- 개발카드 손패 표시 (뒷면 + 사용 버튼)
- 점수 보드 (각 플레이어 VP 실시간)
- 건설 메뉴 (비용 표시 + 건설 가능 여부)
- 거래 UI (드래그 or 버튼으로 자원 선택)
