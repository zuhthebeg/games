# 맞고 · 고스톱 룰 명세 + 구현 현황

화투 맞고(2인) / 고스톱(3·4인). 단일 클라(`game.cocy.io/gostop`) + 서버 MP 엔진(`games-server/functions/games/gostop.ts`).
출처: 나무위키(고스톱/맞고), 한게임/넷마블 가이드, brunch 룰 정리. 지역·합의별 변형 있음 → 아래는 **본 게임 채택 룰**.

범례: ✅구현 ⚠️부분/상이 ❌미구현 — (C)=싱글클라 (S)=서버MP

---

## 1. 카드 / 족보 점수
- 광 5장(1·3·8·11비광·12). **3광 3점**(비광 포함 시 2점)·4광 4점·5광 15점. ✅C ✅S
- 열끗 9장(2·4·5·6·7·8·9·10·12). **끗 5장 1점, 이후 1장당 +1**. ✅C ✅S
- 고도리(2·4·8월 새) 5점. ✅C ✅S
- 띠 10장. **띠 5장 1점, 이후 +1**. 홍단(1·2·3) 3점·청단(6·9·10) 3점·초단(4·5·7) 3점. ✅C ✅S
- 피: **10장 1점, 이후 +1**. 쌍피=2장(11월 쌍피·12월 비쌍피). ✅C ✅S
- **9월 국진 = 쌍피로도 사용**(끗 채우면 열끗, 아니면 쌍피 — 높은 쪽). ✅C ✅S

## 2. 진행 / 고·스톱
- 점수 도달: 맞고 7점 / 고스톱 3점 → 고 또는 스톱. ✅C(7) ✅S(2인7/3인+3)
- **고 배수**: ⚠️ 현재 2^(고-1)(1고×1·2고×2…). 정통은 보통 **1고 +1점·2고 +2점, 3고부터 ×2·4고 ×4**. → 검토 항목.
- 스톱: 종료·정산. ✅
- **고박**: ❌ 고 부른 사람이 다음 자기 차례까지 못 나고 상대가 나면 고 부른 사람이 전부 물기(독박). 미구현.

## 3. 박(배수)
- 피박: 진 사람 피 ≤6 & 이긴 사람 피 점수 → ×2. ✅C ✅S(per-loser)
- 광박: 진 사람 광 0 & 이긴 사람 광 점수 → ×2. ✅C ✅S
- **멍박(멍따)**: ❌ 열끗 7장↑(멍따) → 별도 점수 + 진 사람 열끗 적으면 ×2. 미구현.
- 독박: 맞고에서 고박/광박 등 혼자 뭄(= 위 배수의 1:1 적용). ⚠️ 고박 미구현분 제외 동작.
- 흔들기 ×2 / 폭탄 ×2 / 나가리누적 ×N (아래).

## 4. 특수룰
- 뻑: 바닥1+손1 먹으려는데 뒤집기도 같은 월 → 3장 쌓임, 못 먹음. ✅C ✅S
- 자뻑: 내가 싼 뻑을 내가 먹음 → 피 2장. ✅C ✅S
- 뻑쓸이: 남의 뻑을 먹음 → 피 1장. ✅C ✅S
- **3뻑**: 한 판 뻑 3번 → 즉시 10점 승. ✅C ✅S
- **첫뻑**: ❌ 첫 턴 뻑 보너스(룰별) — 미채택/미구현(보류 가능).
- 따닥: 바닥2+손1+뒤집기1 = 4장 → 피 1장. ✅C ✅S
- 쪽: 바닥0(손패 깐 곳)에 뒤집기 같은 월 → 2장 → 피 1장. ✅C ✅S
- 쓸(싹쓸이): 바닥 비움 → 피 1장. ✅C ✅S
- **폭탄**: 손 같은 월 3장+바닥1 → 3장 투하 4장 먹기 + 피 1장 + **다음 자기 차례 2번 뒤집기전용 턴**. ✅C(방금) ⚠️S(현재 extra_turn 1턴 — flip-only 2턴으로 수정 필요)
- **흔들기**: 손 같은 월 3장(+바닥1) → **흔들기 선언(보여주기)** 후 그 패로 점수 시 ×2. ⚠️C(자동×2, 선언/선택 없음) ⚠️S(자동). → 폭탄과 택일 선택 UI 필요.
- **총통(통)**: ❌ 처음 받은 손패에 같은 월 4장 → 즉시 그 판 승(또는 큰 배수). 미구현.
- 피뺏기 공통: 상대 피 없으면 그냥 못 뺏음(빚 없음). ✅C ✅S. 마지막(더미0) 차례엔 못 뺏음. ✅C

## 5. 나가리
- 아무도 못 남 → 무승부, 선 유지, **다음 판 판돈 ×2 누적**. ✅C ⚠️S(누적 미적용 — 룸 단판 기준)

## 6. 정산
- 점수 × 1만골드 × 판돈배수 × (고·피박·광박·흔들기·폭탄·나가리누적). ✅C
- MP: 제로섬 다인 정산(피박/광박 per-loser). ✅S

## 갭 목록 (잡으로 등록)
1. **흔들기 선언 메커니즘** — 3장+바닥 시 흔들기/폭탄/그냥 선택, ×2(선언 시). (C,S)
2. **폭탄 서버 미러** — flip-only 2턴으로(클라와 동일). (S)
3. **멍박/멍따** — 열끗 7장 멍따 점수 + 멍박 ×2. (C,S)
4. **총통** — 같은 월 4장 즉시 승/배수. (C,S)
5. **고박** — 고 부르고 못 나면 독박. (C,S)
6. **고 배수 룰 검토** — 1·2고 가산 + 3고부터 ×2로 변경 여부. (C,S)
7. **MP 클라 통합** — 멀티 로비+서버상태 렌더러+N인 레이아웃(별도 대형). (C)
8. (보류) 첫뻑 보너스.
