🀄 대만 마작

인간 1명 + CPU 3명
16장 핸드 · 대만 룰

🀄 플레이 방법
🎯 목표
패 16장으로 5몸통 + 1머리를 완성해 화료(和了)!
🔄 기본 순서
1. 뽑기 — 벽에서 패 1장 (내 차례에 자동)
2. 버리기 — 패 클릭해 선택 → 버리기 버튼
3. 반시계 방향으로 4명이 반복
✋ 상대 버린 패 가져오기
상대가 버릴 때 버튼이 뜹니다:
• 화료 — 그 패로 완성 (론 승리)
• 퐁 — 같은 패 3장 (누구 패든)
• 치 — 연속패 3장 (왼쪽 사람 패만)
• 깡 — 같은 패 4장
• 패스 — 넘기기
🃏 몸통 종류
• 순쯔: 연속 3장 (🀇🀈🀉)
• 커쯔: 같은 패 3장 (🀇🀇🀇)
• 깡쯔: 같은 패 4장
• 머리: 같은 패 2장 (딱 1조)
🌸 화패
春夏秋冬·梅蘭菊竹 뽑으면 자동 공개 + 보충패.
자리 일치(정화) 시 추가 점수.
💰 보상
화료 시 타이(台) × 10만 골드
기본 5타이 + 역 합산 (자모·평화·대대화 등)
