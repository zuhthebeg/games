# 대만 마작 — 웹앱 출시 계획

> game.cocy.io/mahjong · 단일 HTML + 공용 lib + games-server(멀티). 본 문서는 정식 출시까지 로드맵.

## 0. 현재 상태 (2026-05-31)
- ✅ 싱글(CPU 3) · 실시간 멀티(server-authoritative) · 골드 베팅 경제
- ✅ 대만 룰 점수엔진(론 정렬버그 수정 완료), 결과 증감/독박 표시
- ✅ 패 콜 음성(녹음 클립, 좌석별 음높이), 깡/퐁/츠/팅/즈모/빵창 효과음, BGM
- ✅ 점수 해설 TTS, 팅 대기패 표시, 확대/축소, 방향별 버림패, 사용자 목소리 덮어쓰기
- ✅ i18n 골격(ko/zh-TW) + 브라우저 언어 자동전환, 0원 참여 가능
- ✅ 설치형 웹앱 토대: manifest + 네트워크우선 SW + OG/SEO 메타
- ⏳ 멀티 턴 페이싱(CPU 턴 즉시처리) 진단 워크플로 진행 중

## 1. 출시 전 필수 (P0)
1. **전체 i18n 완성** — 현재 정적 한국어가 남은 곳: howto 오버레이, 토스트('패 없음','화패…','마이크 권한' 등), 결과창 라벨(독박/방총/잔액/기본5+역), 점수해설 TTS, 보이스 커스터마이저, 지갑 '보유 골드'. → 모든 문자열을 i18n dict로 추출 + 繁體中文 번역. (분량 큼: 워크플로로 일괄 추출·번역 권장)
2. **멀티 페이싱** — 워크플로 진단 결과 반영(서버 AI 턴 사이 딜레이+턴별 상태 emit, 또는 클라 리플레이 큐). 이게 멀티 체감 품질의 핵심.
3. **모바일 QA** — 가로/세로, 노치(safe-area), 확대 180%에서 손패/버림패 레이아웃, iOS Safari 오디오/TTS 정책.
4. **점수엔진 엣지 검증** — 천화/지화/대삼원/사안커 등 고역, 유국, 연속 깡, 본장 누적.

## 2. 마감 품질 (P1)
- PNG 아이콘(192/512/maskable) 생성 → manifest에 추가(현재 SVG만). 스플래시/설치 UX 개선.
- 오프라인 동작 점검(SW 캐시), lib 버전 변경 시 캐시 무효화 확인.
- 사운드 on/off·BGM 기본값, 첫 진입 오디오 언락(모바일 자동재생 정책).
- 결과창/로비 i18n 포함 스크린샷 점검.

## 3. 배포 채널
| 채널 | 방법 | 비용/난이도 | 비고 |
|---|---|---|---|
| **웹/PWA** (now) | game.cocy.io/mahjong, 설치형 | 무료/완료 | 즉시. 안드로이드 '홈 추가' 가능 |
| **cocy 생태계** | `games/apps.json` 등록 + list/home 노출 | 낮음 | 내부 트래픽 유입 |
| **Google Play** | TWA(Bubblewrap)로 PWA 래핑 | 중 (등록비 $25, PNG아이콘·정책 필요) | 설치형 배포·검색 노출 |
| **App Store** | Capacitor/PWA 래핑 | 높음 ($99/년, 심사 까다로움) | 후순위 |

권장 순서: 웹/PWA → cocy 생태계 등록 → (반응 보고) Google Play TWA.
