# 대만 마작 룰 레퍼런스

## 기본 구조
- **144타일**: 만쯔(m1-9×4) + 통쯔(p1-9×4) + 죽쯔(s1-9×4) + 바람(z1-4×4) + 삼원(z5-7×4) + 화패(f1-8×1)
- **16타일 핸드** (일본 마작은 13장, 대만은 16장)
- 친(동가): 배패 후 17장으로 시작 → 첫 버리기
- 화료 조건: **5몸통 + 1머리** (특수패 제외)
- 진행 방향: **반시계** (우→상→좌→하 = p0→p3→p2→p1)
- 치(치): 바로 왼쪽(이전 순서) 플레이어가 버린 패만 가능

## 타일 종류
- **순쯔**: 같은 종류 연속 3 (m1m2m3, p5p6p7 등)
- **커쯔**: 동일 패 3장
- **깡쯔**: 동일 패 4장 (선언 후 보충패 뽑기, 타이 추가)
- **또이쯔**: 동일 패 2장 (머리)

## 화패 처리
- 화패(f1-f8) 뽑으면 즉시 공개 + 보충패 뽑기
- 화패 자리 일치: 좌석 번호와 화패 번호 일치 시 正花 (+2타이)
  - p0(동)=f1,f5 / p1(남)=f2,f6 / p2(서)=f3,f7 / p3(북)=f4,f8
- 춘하추동(f1-f4) 4장 모두 = +2타이
- 매난국죽(f5-f8) 4장 모두 = +2타이
- 화패 없으면 무화 = +1타이
- 화패 8장 모두 = 화만관 +14타이

## 클레임 우선순위
1. **론(화료)** > 퐁/깡 > 치
2. 론은 누구든 가능 (방총 패 선언)
3. 퐁: 상대방 누구든 버린 패로 3장 완성
4. 깡: 상대방 누구든 버린 패로 4장 완성 (명깡)
5. 치: 바로 이전 플레이어가 버린 패로 순쯔 완성
6. 론 동시: 대만 마작은 여러 명 동시 론 허용 (일본과 다름)

## 역(台) 전체 목록

### 1타이
- **츠모(自摸)**: 벽에서 직접 뽑아 화료
- **멘젠(門前)**: 오픈패(치/명퐁/명깡) 없이 화료 (론 포함)
- **단기대기(單騎)**: 마지막 1장만 기다리는 형태 (머리 완성 대기)
- **무자(無字)**: 자패(z1-z7) 전혀 없음
- **영상개화(嶺上開花)**: 깡 선언 후 보충패로 화료
- **해저(海底)**: 마지막 남은 벽 패로 자모
- **바깥깡(明槓)**: 상대 버린 패로 깡 (1타이 추가)

### 2타이
- **안깡(暗槓)**: 손에서 4장 모아 깡 (2타이 추가)
- **핑후(平和)**: 모든 몸통 순쯔 + 역패 아닌 머리 + 양면대기
- **삼안커(三暗刻)**: 비공개 커쯔 3개
- **전구인(全求人)**: 모든 몸통 울어서 만들고 론으로 화료
- **춘하추동**: f1+f2+f3+f4 모두 보유
- **매난국죽**: f5+f6+f7+f8 모두 보유
- **친 연장**: 친 연장 1회당 +2타이

### 3타이
- **멘젠츠모(門前自摸)**: 멘젠 상태에서 자모

### 7타이
- **또이또이(碰碰胡)**: 모든 몸통 커쯔 (5커쯔+1머리)
- **사안커(四暗刻)**: 비공개 커쯔 4개
- **소삼원(小三元)**: 삼원패(z5z6z7) 중 2종 커쯔 + 1종 머리
- **소사희(小四喜)**: 바람패(z1z2z3z4) 중 3종 커쯔 + 1종 머리

### 14타이
- **오안커(五暗刻)**: 비공개 커쯔 5개 (깡 포함)
- **대삼원(大三元)**: 삼원패 3종 모두 커쯔
- **대사희(大四喜)**: 바람패 4종 모두 커쯔
- **화만관(花滿貫)**: 화패 8장 모두
- **천화(天和)**: 친이 첫 배패로 즉시 화료
- **지화(地和)**: 친의 첫 버리기로 다른 플레이어 화료

### 기타 점수
- **무화**: 화패 없음 +1
- **정화(正花)**: 자리 일치 화패 각 +2
- **역패 커쯔**: 자풍/장풍 일치 +2, 불일치 +1, 삼원패 커쯔 +1(백발중)
- **친 장가**: 친 플레이어 화료 시 +1
- **기본 점수**: 5타이 기본
