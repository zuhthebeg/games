# 패 콜(打牌) 보이스 녹음 가이드

게임이 `voice/<id>.mp3` 를 재생합니다. 긴 녹음을 무음 기준으로 자동 컷 → 정규화 → mono mp3로 구워서 아래 파일명으로 떨어뜨립니다.

## 넣을 경로 (raw 원본)
- WSL: `/mnt/c/Users/user/games/mahjong/voice/raw/`
- Windows: `C:\Users\user\games\mahjong\voice\raw\`

## 녹음 규칙
- 각 발음 사이 **0.5초 이상 무음**(쉬는 간격 일정하게).
- 같은 거리/볼륨, 조용한 환경. 포맷 무관(wav/m4a/mp3), mono OK.
- **순서 절대 지키기**, 빼먹지 말기. 헷갈리면 그 파일만 다시.
- 발음 언어/스타일(華語·台語·한국어 읽기)은 자유.

> ⚠ honorfx 순서 주의: 風패 **東→南→西→北**, 三元패 **白→發→中** (게임 ID: z1東 z2南 z3西 z4北 z5白 z6發 z7中), 그 뒤 효과음 **후 퐁 츠**.
> ⚠ etc 순서 주의: **깡 안깡 보충패 팅 즈모 빵창** 정확히 이 순서 6개.

- 花패(春夏秋冬梅蘭菊竹)는 자동 오픈이라 콜 없음 → 녹음 불필요.
- 자동 분할 후 컷 개수가 표와 다르면 그 파일만 다시 녹음 요청합니다(임의 라벨링 안 함).

## fx 발동 시점 (게임 내)
- fx-hu: 론(방총) 완료 / fx-tsumo: 즈모(자력) 완료 / fx-bangchong: 누가 방총당했을 때(완료 0.65초 뒤)
- fx-pon·chi·kan·ankan: 퐁/츠/깡/안깡 선언 시 / fx-rinshan: 깡 후 보충패 뽑을 때 / fx-tenpai: 텐파이(팅) 진입 시

## 사용자 목소리 덮어쓰기(개별 플레이어용)
설정 → 🎙 "내 목소리로 덮어쓰기" 에서 누구나 위 소리들을 자기 목소리로 녹음해 덮어쓸 수 있음.
브라우저 로컬(IndexedDB) 저장이라 본인 기기에만 적용. ↺ 또는 "전체 초기화" 시 위 기본 녹음으로 복귀.
