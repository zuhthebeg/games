# 카탄 맵 설계

## 기본 맵 (현재 구현)
- 19 헥스 (3-4-5-4-3 배열)
- 자원: 벽돌3, 나무4, 양4, 밀4, 광석3, 사막1
- 숫자: 2,3,3,4,4,5,5,6,6,8,8,9,9,10,10,11,11,12
- 밸런스: 6/8 인접 불가, 같은 숫자 인접 불가
- 항구: 일반4(3:1), 전문5(각 자원 2:1)
- 3~4인

## 확장 맵 1: 5~6인 확장
- **30 헥스** (4-5-6-5-6-5-4 배열, 외곽 바다 포함)
- 추가 자원: 벽돌2, 나무2, 양2, 밀2, 광석2, 사막1 → 총 자원 타일 30개
- 추가 숫자 토큰: 2,3,4,5,6,8,9,10,11,12 (각 1개씩 추가)
- 특수 규칙: 특별 건설 페이즈 (자기 턴 아닐 때도 건설 가능)
- 5~6인 지원

## 확장 맵 2: 미니 맵 (2인 전용)
- **7 헥스** (1-2-3-2-1 → 실제로 중앙+6개 = 7개)
- 자원: 벽돌1, 나무1, 양1, 밀1, 광석1, 사막1, 금광1
- 숫자: 3,4,5,9,10,11
- 빠른 게임: 6VP로 승리
- 금광: 아무 자원 1개 선택 획득

## 확장 맵 3: 해양 (Seafarers 간소화)
- **37 헥스** (기본19 육지 + 18 바다/섬)
- 바다 타일: 배 건설 가능 (도로와 동일 비용 + 양 대신 나무)
- 섬: 2~3개 작은 섬에 추가 자원
- 금광 타일: 주사위 맞으면 아무 자원 선택
- 특수: 선착장 발견 보너스 (+1VP)

## 구현 우선순위
1. ✅ 기본 맵 (완료)
2. 📋 미니 맵 (2인 전용, 빠른 구현)
3. 📋 5~6인 확장
4. 📋 해양 확장
